# My Event 

## Instructions
To test it online
1. Go to https://events-wanabe.herokuapp.com/

To test it in local server

1. Clone from Github
2. ```bundle install```
3. ```rails db:create```
4. ```rails db:migrate```
5. Go to Go to http://localhost:3000

## Ruby & Rails version

- Ruby 2.5.1

- Rails 5.2.4

## Database

- PostGreSQL

## Functionalities

- Create an event
- See events
- Create user
- See user
- sign up for an event
